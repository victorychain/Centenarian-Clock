
# Centenarian Clocks

## Install requirments

```bash
pip install -r requirments.txt
```

## Usage

### Data
Put data in a CSV file such as ./datMeth.csv in the format that each columns is one sample, each row is a CpG. 

### Terminal
```bash
$ python main.py --datMeth ./datMeth.csv --output_dir ./
# for help
$ python main.py -h
```
